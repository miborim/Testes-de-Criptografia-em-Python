## 📝 **Descrição**

Este repositório contém a implementação e análise de SHA-256 e AES-256 em Python, utilizando diferentes bibliotecas. O objetivo é comparar o desempenho e a segurança dessas funções criptográficas por meio de 10 testes distintos, registrando os resultados em uma tabela e elaborando uma análise comparativa.
